# ShipmentTracker
Shipment Tracker project for the DBMS course.
### To start client:
`cd client`<br>
`npm init(first time)`<br>
`yarn start`<br>
### To start server:
`cd server`<br>
`npm init(first time)`<br>
`nodemon server`
#### Add a .env file to your server folder and add the database password.
